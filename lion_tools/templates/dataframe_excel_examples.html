to_excel()

Creates an Excel worksheet from Spark DataFrames with extended formatting functionality.

Features

  • Multiple sheets by creating a sheet per dataframe
  • Formatting of values
  • Background colors of columns
  • Set column width to longest value (automatic)
  • Autofilter (automatic)

Parameters

  • *dfs: list of dataframes or dictionaries with more details. See examples for the workings of this parameter.
    Format Definition: 
    dict(
    df=dataframe,
    name=sheet name,
    column_options=[
        'column name': dict(
            width=1-...,
            format='format code, see below',
            bg_color='color name or #code',
            other options, see https://xlsxwriter.readthedocs.io/format.html#format
            for details
        ),
        ...
    ],
    autofilter=True/False,
    index=True/False,
    autofit=True/False,
    freeze_first_row=True/False,
    border=0-13,
    border_color='color name or #code',
    num_format='format code, see below, or Excel native format',
),
dict(
    ... -> additional entry per dataframe=sheet
),

Usage Examples

Example 1: Standard Export

Export the dataframe df to file my_file.xlsx in the standard export directory (/Workspace/Shared/excel_exports/)

to_excel(df, 'my_file.xlsx')

Example 2: Subdirectory Export

Export to a subdirectory of the standard export directory.

to_excel(df, 'mdleeuw/my_file.xlsx')

Example 3: Full Path

Export to a full path in the Workspace.

to_excel(df, path='/Workspace/Teams/BestTeam/some_sub_sir/my_file.xlsx')

Example 4: Column Colors

Set the color of some of the columns.

to_excel(
    {
        'df': df,
        'column_options': {
            'article_id': {'bg_color': 'yellow'},
            'article_desc': {'bg_color': '#FFF0F0'},
        },
    },
    file_name='my_file.xlsx'
)

Example 5: Numeric Formatting

Set format of all numeric columns to have 0 decimals and thousands separator.

to_excel(
    {
        'df': df,
        'num_format': 'number0',
    },
    file_name='my_file.xlsx'
)

Example 6: Specific Column Formatting

Set format of columns to have thousands separator, one column with 0 decimals and one column with a euro sign and 2 decimals.

to_excel(
    {
        'df': df,
        'column_options': {
            'jobber_end_qty_at_financial_cost_price_amt': {'format': 'number0'},
            'jobber_end_qty_at_csp_amt': {'format': 'euro2'},
        },
    },
    file_name='my_file.xlsx'
)

Example 7: Multiple DataFrames to Sheets

Export 2 dataframes to 2 sheets in the same Excel file.

to_excel([voorbeeld_df, voorbeeld2_df], file_name='my_file.xlsx')

# or identical:
to_excel([{'df': voorbeeld_df}, {'df': voorbeeld2_df}], file_name='my_file.xlsx')

Example 8: Named Sheets

2 named sheets.

to_excel([
    {'df': voorbeeld_df, 'name': 'Revenue'},
    {'df': voorbeeld2_df, 'name': 'Inventory'}
], file_name='my_file.xlsx')

Example 9: Mixed Options

First sheet with options, second sheet without options.

to_excel([
    {
        'df': voorbeeld_df, 'name': 'Revenue', 'column_options': {
            'revenue_amount': {'bg_color': 'yellow', 'format': 'euro0'},
            'quantity': {'bg_color': '#FFF0F0'},
        }
    },
    voorbeeld2_df,
], file_name='my_file.xlsx')
